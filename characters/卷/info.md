# 中文字源紀錄：卷 (juǎn / juàn)

## 1. 基本資訊
- **字形**：卷
- **讀音**：juǎn (動詞，如捲曲) / juàn (名詞，如書卷)
- **構造**：從「卩」（跪坐的人），從「𠹔」（兩隻手在上方撥弄或捲曲之形）。

## 2. 字源演變
- **甲骨文/金文**：上部為兩隻手，下部為一個跪坐的人（卩）。形象地表達了兩手將人或物彎曲、捲起的動作。
- **小篆**：結構更加整齊化，其核心意義依然是「屈」與「收羅」。
- **演變邏輯**：屈體（如人跪坐） -> 捲曲物體 -> 紙張發明後的「卷軸」 -> 冊籍。

## 3. 核心義項
1. **彎曲、捲起**：如「捲土重來」、「捲曲」。
2. **書本、冊籍**：如「手不釋卷」、「卷宗」。
3. **優美、彎曲的樣貌**：如「卷髮」。

## 4. 繪製風格分析 (Style Analysis - 延續 wan 系列規範)
- **藝術風格**：傳統中國工筆重彩與現代數位繪法結合。
- **色彩基調**：溫潤的古籍紙張色（米黃）、深沈的小葉紫檀色（捲軸桿）、沈穩的墨綠或靛藍（封皮）。
- **光影與氛圍**：安靜的書齋光影，微弱的燭光或窗邊自然光，營造出時光流轉的寧靜感。
- **構圖特徵**：建議採用水平或垂直的動態線條，表現捲軸展開或收攏的張力。
- **線條細節**：像髮絲般細膩的線條，表現紙張或織物的褶皺與質感。

## 5. 視覺開發建議 (Prompt 構思)
- **第一階段 (素描)**：聚焦於雙手捲起一卷古籍的解剖結構，背景簡約。
- **第二階段 (彩色)**：加入古色古香的書齋環境，光線聚焦在捲軸中心的書法字上。
