# 中文字源紀錄：嘴 (zǔi)

## 1. 基本資訊
- **字形**：嘴
- **讀音**：zǔi
- **構造**：形聲字。從「口」，「觜」（zī / zuǐ）聲。

## 2. 字源演變
- **本字 (觜)**：「觜」原指鳥類頭上的毛（冠），後引申為鳥喙（鳥嘴）。
- **演變邏輯**：為了區分鳥喙與人類或動物的口部，後世增加了「口」字旁，形成「嘴」字。
- **核心意象**：進食、言語、呼吸的器官；也引申為物體突出的部分（如：壺嘴）。

## 3. 核心義項
1. **口部**：人類或動物的嘴巴。
2. **說話/言語**：如「七嘴八舌」、「親嘴」。
3. **器物突出處**：如「瓶嘴」、「煙嘴」。

## 4. 繪製風格分析 (Style Analysis - 延續 wan/juan 系列規範)
- **藝術風格**：工筆重彩與虛實水墨結合。
- **色彩基調**：溫潤的朱砂色（唇色）、淡雅的宣紙背底、略帶乾筆的焦墨（輪廓）。
- **光影與氛圍**：聚焦於口部細微的動作或吐納出的煙雲/氣息。背景宜留白，營造出空靈感。
- **構圖特徵**：特寫鏡頭。重點描繪嘴部的線條美，可結合「氣」的流動（如：墨韻升騰）。
- **線條細節**：唇紋的細膩處理，與周圍暈染開的水墨形成鮮明對比。
