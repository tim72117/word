# 中文字源紀錄：耳 (ěr)

## 1. 基本資訊
- **字形**：耳
- **讀音**：ěr
- **構造**：象形字。

## 2. 字源演變
- **甲骨文**：完全是耳朵的形狀，描繪出耳廓與耳垂的特徵。
- **金文**：延續甲骨文，線條變得較為平整。
- **小篆**：《說文解字》：「主聽也。象形。凡耳之屬皆从耳。」
- **演變邏輯**：從具象的耳朵形狀，逐漸簡化為現在的漢字結構，保留了耳廓的輪廓感。

## 3. 核心義項
1. **聽覺器官**：如：耳朵、耳聰目明。
2. **形狀像耳朵的東西**：如：木耳、雲耳、鍋耳。
3. **位置在兩旁的**：如：耳房、耳語。
4. **虛詞**：同「而已」（如：無他，但手熟爾 -> 耳）。

## 4. 繪製風格分析 (Style Analysis)
- **藝術風格**：工筆重彩，呈現玉石般的溫潤感或宣紙般的輕盈感。
- **色彩基調**：溫潤的肉粉色、和諧的象牙白、以及代表靈敏聽覺的微弱金邊。
- **光影與氛圍**：通透、輕靈。強調邊緣光的表現，展現耳朵的透光性。
- **構圖特徵**：流線型、動態。耳朵微傾，彷彿正在捕捉周遭的細微聲響。

## 5. 視覺開發建議 (Prompt 構想)
- **畫面中心**：一隻如玉雕般細膩、輪廓優美的耳朵。
- **細節**：半透明的質感、如流水般的曲線、以及耳郭內隱約可見的細節。
