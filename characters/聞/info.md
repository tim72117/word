# 中文字源紀錄：聞 (wén)

## 1. 基本資訊
- **字形**：聞
- **讀音**：wén
- **構造**：形聲字。從耳，門聲。

## 2. 字源演變
- **甲骨文**：通常表現為一個側坐的人形，手放在耳朵旁，強調「聽」的動作。
- **金文**：增加了「門」的結構。外圍是「門」，門內放一個「耳」。形象地表達了在門口傾聽，或者聲音穿門而出的意象。
- **小篆**：《說文解字》：「知聲也。从耳，門聲。」
- **演變邏輯**：從單純的「耳」部動作，演化為「門內有耳」，象徵聽取外部訊息、傳播名聲。

## 3. 核心義項
1. **聽見、聽到**：感官對聲音的接收（如：耳聞、聞風而逃）。
2. **聽說、知道**：獲取資訊（如：新聞、見聞）。
3. **名聲、名望**：被眾人所知（如：聲聞、顯聞）。
4. **用鼻子嗅**：後來引申出的嗅覺意義（如：聞香）。

## 4. 繪製風格分析 (Style Analysis)
- **藝術風格**：延續《中文字源》系列的「工筆重彩」與「水墨渲染」。
- **色彩基調**：古樸的木質色（門）、肉粉或玉白（耳）、以及代表資訊流動的淡淡青煙色。
- **光影與氛圍**：一種「隔牆有耳」或「門縫窺音」的神秘與專注感。光線應從門縫中透出，照亮耳朵的一側。
- **構圖特徵**：畫面中心是一座古色古香的朱漆大門或雕花木門，門扉微啟。一隻造型優雅的耳朵優雅地懸浮或貼近門縫處。背景可以有一些象徵聲音波動的細膩水波紋或雲氣。

## 5. 視覺開發建議 (Prompt 構想)
- **畫面中心**：一扇極具質感的中國傳統大門（門），門軸處透出微光。一隻如蟬翼般通透、具備工筆畫線條感的耳朵（耳）位於門縫間。
- **元素結合**：門的剛硬（直線、木紋）與耳的柔軟（曲線、血脈感）形成對比。
- **背景**：深色的環境色，強調門縫透出的那一線光。
