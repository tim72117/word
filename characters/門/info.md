# 中文字源紀錄：門 (mén)

## 1. 基本資訊
- **字形**：門
- **讀音**：mén
- **構造**：象形字。

## 2. 字源演變
- **甲骨文**：像兩扇門扉的樣子，有門框、門楣和兩扇對開的門板。
- **金文**：基本上延續甲骨文的形制，線條更加整齊圓潤。
- **小篆**：《說文解字》：「聞也。从二戶，象形。」這裡的「聞」指門戶相合，兩戶為門。
- **演變邏輯**：從實體的雙扇門，演變為代表出入口、家族、類別的抽象符號。

## 3. 核心義項
1. **建築物的出入口**：如：大門、校門。
2. **形狀像門的東西**：如：門牙、閥門。
3. **家族、家道**：如：門當戶對、名門。
4. **類別、學派**：如：部門、學門、師門。

## 4. 繪製風格分析 (Style Analysis)
- **藝術風格**：工筆重彩，強調木質紋理與古建築結構美。
- **色彩基調**：沈穩的朱砂紅、斑駁的古木棕、以及點綴的青銅門環綠。
- **光影與氛圍**：宏偉、莊嚴。光線從門的正上方投射，強調門楣的體積感。
- **構圖特徵**：正直、對稱。採用正面構圖，展現兩扇大門並立的姿態。

## 5. 視覺開發建議 (Prompt 構想)
- **畫面中心**：一對氣勢恢宏的傳統中式朱漆大門，門上有古樸的青銅獸首門環（椒圖）。
- **細節**：木門的裂紋、層層堆疊的油漆感、金屬的銅綠。
