# 中文字源紀錄：鼻 (bí)

## 1. 基本資訊
- **字形**：鼻
- **讀音**：bí
- **構造**：形聲字。從「自」（鼻子的象形），「畀」聲。

## 2. 字源演變
- **本字 (自)**：「自」在甲骨文中是鼻子的正面象形，有鼻樑和鼻翼。
- **演變邏輯**：古人說到自己時常指著鼻子，因此「自」引申為「自己」。為了區分，後世在「自」下方加上聲符「畀」，形成現在的「鼻」字。
- **核心意象**：呼吸的門戶、生命之氣的起點（「鼻祖」之意由此而來）。

## 3. 核心義項
1. **呼吸與嗅覺器官**：如「鼻子」、「鼻息」。
2. **創始/第一**：如「鼻祖」(最早的創始人)。
3. **器物突出部分**：如「門鼻兒」。

## 4. 繪製風格分析 (Style Analysis)
- **視覺重心**：鼻樑的挺拔感與鼻翼的弧度。應呈現出一種「生命氣息」流動的感覺。
- **裝飾元素**：可結合「雲煙」或「仙氣」，象徵從鼻子呼出的「氣」。
- **色彩基調**：古雅的宣紙色底，局部使用極淡的肉色，主要以墨色勾勒。

## 5. 視覺開發建議 (Prompt 構思)
- **第一階段 (素描)**：聚焦於鼻子的立體結構，線條應挺拔有力，背景帶有淡淡的氣流曲線。
- **第二階段 (彩色)**：強化宣紙質感，加入細膩的工筆重彩，讓「氣」的部分呈現出水墨煙雲的效果。
